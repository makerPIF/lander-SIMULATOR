# 낙하산 크기 계산 가이드

> SRC 로켓·탐사 프로그램 / 호박공장 메이커스페이스  
> 낙하산 설계·제작을 위한 물리 공식 및 변수 해설서

---

## 1. 핵심 물리 공식

낙하산의 크기는 **항력(Drag Force) = 중력(Weight)** 조건에서 결정됩니다.

```
D = ½ · ρ · Cd · A · v²  =  m · g
```

따라서 필요한 펼침 면적 A는:

```
A = (2 · m · g) / (ρ · Cd · v²)
```

| 기호 | 의미 | 단위 |
|------|------|------|
| A | 낙하산 펼침 면적 | m² |
| m | 페이로드 질량 | kg |
| g | 중력 가속도 | m/s² |
| ρ | 공기 밀도 | kg/m³ |
| Cd | 항력 계수 | 무차원 |
| v | 목표 착지 속도 | m/s |

**원형 낙하산 지름:**

```
d = √(4A / π)
```

## 3. 낙하산 형태별 항력 계수 (Cd)

| 형태 | Cd | 특징 | 권장 용도 |
|------|-----|------|----------|
| 평면형 (Flat) | 1.05 | 단순 원형. 제작 쉬움 | 저속 모델 로켓, 교육용 |
| 반구형 (Hemispherical) | 0.75 | 안정적, 표준형 | 일반 로켓, 드론 페이로드 |
| 십자형 (Cross) | 1.00 | 횡방향 이동 억제 | 정밀 착지, 사운딩 로켓 |
| 환형 (Annular) | 0.85 | 중앙 구멍, 진동 감소 | 중속, 안정성 우선 |
| 리본형 (Ribbon) | 0.55 | 고속 안정화 특화 | 드로그 낙하산, 초음속 |
| X-form | 0.90 | 사각 변형, 군사·탐사 | 특수 임무 |

> **드로그 낙하산 + 메인 낙하산 이중 시스템**: 드로그는 리본형(Cd 0.55)으로 고속에서 안정화 후, 저고도에서 메인 낙하산(반구형 등)을 전개.

## 4. 안정적 착륙을 위한 추가 설계 변수

### 4-1. 안전계수 (Safety Factor, SF)

```
A_설계 = A_이론 × SF
```

| 용도 | 권장 SF |
|------|--------|
| 교육용 모형 | 1.2 |
| 일반 무인 탑재체 | 1.3 ~ 1.5 |
| 정밀 장비 탑재 | 1.5 ~ 2.0 |
| 유인 시스템 | 2.0 ~ 3.0 |

---

### 4-2. 개산 충격 계수 (Opening Shock Factor, Cx)

전개 순간 발생하는 충격력은 정적 항력보다 훨씬 큽니다.

```
F_충격 = Cx · ½ · ρ · Cd · A · v_전개²
```

| 전개 방식 | Cx 범위 |
|----------|--------|
| 자유 전개 (Free deployment) | 1.5 ~ 2.5 |
| 슬라이더 장착 (Slider) | 1.1 ~ 1.3 |
| 리핑 전개 (Reefing) | 1.0 ~ 1.2 |

> **낙하산 줄 인장강도 요구값** = F_충격 × 현수선 수로 나눠 각 줄의 최소 강도 계산

---

### 4-3. 현수선 (Suspension Lines)

**현수선 수 (N):** 일반적으로 낙하산 지름(cm)의 절반 수 권장

```
N ≥ d(cm) / 2
```

예: 지름 60 cm 낙하산 → 최소 30 가닥 (실용 최소 8~16 가닥)

**현수선 길이 비율 (L/D):**

```
L = k · d  (k = 1.0 ~ 1.5 권장)
```

| L/D 비율 | 특성 |
|----------|------|
| 0.8 이하 | 진동 심함, 불안정 |
| 1.0 | 기본 안정 |
| 1.2 ~ 1.5 | 진동 최소, 권장 범위 |
| 2.0 이상 | 과도히 길어 실용성 저하 |

---

### 4-4. 벤트 홀 (Vent Hole)

캐노피 꼭대기의 구멍으로 진동 억제 및 안정성 향상.

```
벤트 비율 = D_벤트 / D_낙하산 × 100 (%)
```

| 벤트 비율 | 효과 |
|----------|------|
| 0 % (없음) | 진동 심함 (봉투 효과) |
| 5 ~ 10 % | 안정성 향상, Cd 약간 감소 |
| 15 ~ 20 % | 매우 안정, Cd 5~8% 감소 |
| 25 % 이상 | Cd 급감, 착지 속도 증가 위험 |

> 벤트 적용 시 실효 Cd는 약 3~8% 감소하므로 면적을 해당 비율만큼 보정

---

### 4-5. 바람 영향 및 드리프트 거리

착지 지점 예측을 위한 드리프트 계산:

```
드리프트(m) = 바람속도(m/s) × 낙하시간(s)
낙하시간(s) = 전개 고도(m) / 착지 속도(m/s)
```

착지 속도 실효값 (바람 포함):

```
v_실효 = √(v_수직² + v_바람²)
```

---

### 4-6. 최대 허용 착지 속도 기준

| 페이로드 유형 | 권장 착지 속도 |
|------------|--------------|
| 견고한 금속 구조물 | ≤ 10 m/s |
| 일반 전자장비 | ≤ 7 m/s |
| 카메라·광학기기 | ≤ 5 m/s |
| 달걀 낙하 실험 | ≤ 3 m/s |
| 유인 시스템 기준 | ≤ 6 m/s (군용) / ≤ 5 m/s (민간) |

| 나일론 립스탑 (70D) | 약 80 g/m² | 내구성 우수 | 중형 탑재체 |

---

### 5-2. 원형 평면형 낙하산 제작 순서

1. **패턴 절단**: 계산된 지름 + 시접(2~3 cm) 여유 포함하여 원형 재단
2. **고어 분할 (선택)**: 더 좋은 형태를 위해 원을 8~16조각(고어)으로 분할 재단 후 봉제
3. **벤트 홀 재단**: 상단 중앙에 계산된 벤트 지름의 원형 구멍 절취, 테두리 보강
4. **현수선 부착점 표시**: 캐노피 가장자리를 N등분하여 부착점 표시
5. **테이프 보강**: 낙하산 가장자리 및 현수선 부착점에 방사형 보강 테이프 봉제
6. **링 링크 부착**: 금속 링 또는 슬라이드 링을 현수선 집결점에 부착
7. **현수선 연결**: 계산된 길이(d × 1.2)로 절단, 캐노피-링 링크 연결
8. **중앙 하네스 연결**: 링 링크 → 탑재체 하네스/아이볼트 연결

---

### 5-3. 패킹 (접기)

| 방법 | 특징 |
|------|------|
| Z-접기 | 가장 일반적. 펼침 신뢰성 높음 |
| 방사형 접기 | 원형 낙하산 전용, 균일 전개 |
| 번들 방식 | 단순하지만 엉킬 위험 |

> 패킹 후 지름 기준 수납 원통 지름:  
> `d_통 ≈ √(A_낙하산 × t_원단) / π` (경험식 참고용)

## 6. 계산 예시

**조건:** 지구 해수면, 질량 0.5 kg, 착지 속도 5 m/s, 반구형 낙하산, SF = 1.3

```
A = (2 × 0.5 × 9.81) / (1.225 × 0.75 × 5²)
  = 9.81 / 22.97
  = 0.427 m²

A_설계 = 0.427 × 1.3 = 0.555 m²

d = √(4 × 0.555 / π) = √(0.707) = 0.841 m ≈ 84 cm
```

벤트 홀 (10%): `d_벤트 = 84 × 0.10 = 8.4 cm`  
현수선 수: `N = 84 / 2 = 42 → 실용적으로 16가닥`  
현수선 길이: `L = 84 × 1.2 = 100.8 cm ≈ 101 cm`

## 7. 자주 묻는 질문

**Q. 왜 화성에서 낙하산이 어려운가?**  
대기 밀도가 지구의 약 1/60 수준(0.020 kg/m³)이므로, 같은 질량·속도 조건에서 약 60배 큰 낙하산이 필요합니다. NASA의 퍼시비어런스 로버는 지름 21.5 m의 초대형 낙하산을 사용했습니다.

**Q. 낙하산이 펼쳐지지 않으면?**  
드로그 낙하산(소형 파일럿 낙하산)으로 메인 낙하산을 강제 전개시키는 이중 시스템을 적용합니다.

**Q. 교육용 달걀 낙하 실험에서 적정 크기는?**  
달걀 + 용기 = 약 100 g, 목표 속도 ≤ 3 m/s, 지구 해수면, 반구형(Cd 0.75), SF 1.3:

```
A_설계 = (2 × 0.1 × 9.81) / (1.225 × 0.75 × 9) × 1.3 ≈ 0.310 m²
d ≈ 63 cm
```
