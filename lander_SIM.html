📱 스마트폰/갤럭시탭 업로드 방법
방법 1: ArduinoDroid (가장 추천)
Google Play → ArduinoDroid 검색·설치
USB OTG 케이블로 아두이노 연결
코드 붙여넣기 → 컴파일 → 업로드 버튼

방법 2: Arduino Web Editor
크롬에서 create.arduino.cc 접속
새 스케치 생성 → 코드 붙여넣기 → 업로드

방법 3: Chrome Web Serial (갤럭시탭)
chrome://flags → Experimental Web Platform 활성화
USB OTG 연결 후 Web Serial API 직접 사용

방법 4: WiFi OTA (ESP32 보드)
아두이노 우노 → ESP32 교체 시 무선 업로드 가능
🔌 배선 연결도 (Wiring Diagram)
HC-SR04 VCC → 5V | GND → GND
HC-SR04 TRIG → Pin 9
HC-SR04 ECHO → Pin 10
서보1(앞-왼) → Pin 3
서보2(앞-오) → Pin 5
서보3(뒤-왼) → Pin 6
서보4(뒤-오) → Pin 11
MPU6050 SDA → A4 | SCL → A5
상태 LED(초록) → Pin 13 + 220Ω
경고 LED(빨강) → Pin 12 + 220Ω
📱 이 페이지 QR 코드
스마트폰 카메라로 스캔하면 바로 접속됩니다.
🔗 직접 URL 입력하여 QR 생성
GitHub Pages 배포 후 URL을 입력하세요.
📤 GitHub Pages 배포 5단계
① GitHub 로그인
github.com 접속 → 로그인 (또는 회원가입)

② 새 저장소(Repository) 만들기
우상단 + 버튼 → New repository
이름 입력: lander-gnc → Public 선택
→ Create repository 클릭

③ 파일 업로드
Add file → Upload files 클릭
다운로드한 index.html 드래그 앤 드롭
→ Commit changes 클릭

④ GitHub Pages 활성화
Settings 탭 → Pages 메뉴
Source: Deploy from a branch → main → / (root)
→ Save 클릭 (1~3분 후 URL 생성)

⑤ QR 코드 생성
생성된 URL: https://아이디.github.io/lander-gnc/
위 입력칸에 붙여넣기 → QR 생성 → 스캔!
